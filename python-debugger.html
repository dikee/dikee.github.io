Python Debugger (pdb)

  • Mon 12 October 2015
  • ·
  • Python
  • ·
  • python, pdb

When I'm trying to learn how a program works, I often reach for pdb.

pdb is a standard library that allows you to drop a breakpoint into your code and step through line-by-line.

You can set a breakpoint by entering import pdb; pdb.set_trace() at the desired spot in your codebase.

Running code will stop at the breakpoint and allow you to interact via the terminal.
